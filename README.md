# B. Maruthi — Professional Portfolio

A premium, highly interactive, and minimalist portfolio website showcasing core competencies in **Computer Science, Data Science, Machine Learning, and Generative AI**.

Live repository: [https://github.com/maruthi2005-collab/maruthi-portfolio](https://github.com/maruthi2005-collab/maruthi-portfolio)

---

## 🌟 Key Features

- **Ambient Glow Background**: Sleek, modern floating glowing blobs with CSS keyframe float animations that create an immersive ambient design.
- **Custom Interactive Cursor**: A bespoke magnetic cursor with an inner dot and smooth lag-following outer ring that reacts dynamically when hovering over clickable elements.
- **Responsive Fluid Layout**: Fully optimized and styled using modern CSS variables, grid systems, and flexbox structures for desktop, tablet, and mobile displays.
- **Interactive Inquiry Engine**: A premium contact interface that features quick topic selector chips (Data Science, AI/ML Roles, Research, Say Hi) that reactive-sync to pre-populate inquiries.
- **Professional Academic Timeline**: Clean timeline presenting Bachelor of Technology (B.Tech) CSE details, certifications, and specialized system configurations.
- **Micro-Animations & Smooth Scroll**: Intersection Observer API integrations to trigger smooth CSS animations as sections enter the viewport.

---

## 🛠️ Tech Stack & Architecture

- **Structure**: Semantic HTML5 markup ensuring robust SEO foundations and high accessibility.
- **Styling**: Vanilla CSS3 custom properties (CSS variables), HSL-curated gold/sand accent palettes, and responsive font structures.
- **Interactions**: Native Vanilla JavaScript handling custom pointer events, scrolling transitions, and smart form state configurations.
- **Tailwind Ecosystem**: Precompiled stylesheet (`portfolio.txt`) containing Tailwind CSS v4.3.0 frameworks ready to scale if needed.

## 🚀 Local Development

To run and preview the portfolio locally, you can use the lightweight Python development server or any Node.js HTTP server.

### Option A: Using Python (Installed & Configured)
Run the following command in the project directory:
```bash
python -m http.server 8000
```
Then, open your web browser and navigate to:
[http://localhost:8000](http://localhost:8000)

### Option B: Using Node.js / npx
Run the server with zero installation:
```bash
npx http-server -p 8000
```
